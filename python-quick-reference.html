Python Quick Reference

Lambda function simplified using comprehension

            >>> lookup = {'A': 'aaa', 'B': 'bbb', 'C': 'ccc'}
            >>> keys = ['A', 'B', 'Z']
            >>> result = {key:lookup.get(key) for key in keys}
            >>> result
            {'Z': None, 'A': 'aaa', 'B': 'bbb'}

Lambda function simplified using dictionary map

            >>> lookup = {'A': 'aaa', 'B': 'bbb', 'C': 'ccc'}
            >>> keys = ['A', 'B', 'Z']
            >>> result = dict((lambda x: (x,lookup[x]) if x in lookup else (x,None))(key) for key in keys)

Same result using map

        result = dict(map((lambda x: (x, lookup[x]) if x in lookup else (x,None)), keys))
